Authors: Noah Steinberg and James Fitzgerald

Written on OSX tested on Chrome

Setup:  launch the server.js from the Server folder (This is the server and thermostat);
		launch the client.js from the Client folder (This is the furnace);

Usage:   Direct your webbrowser to localhost:3000
		 Use the buttons on the web page to control the thermostat
Note: the default thermostat is 20 degrees
